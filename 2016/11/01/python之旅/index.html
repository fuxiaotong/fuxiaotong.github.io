python之旅

python 印象

python给我的第一印象是一个非常方便的语言，它似乎能尽可能缩短你的代码量，让你更加专注于你的逻辑，让你能更加轻松更加容易地实现你的idea。有了这个神器，你能快速完成一个用C++、java这些语言很难完成的任务，话不多说，让我们进入这个神圣的殿堂，翻开python的第一页书吧！

入门

我们来看看 python 的操作符，print 3**2 结果是9,也就是说两个星号表示的是乘方
print 2 > 4 返回的是False
在表示不等的时候最好用!=
python 只能是以字母或者下划线结尾的，也是大小写敏感的。python也是一门动态语言，也就是说，它不需要事先声明变量的类型，而是在赋值的那一刻进行初始化的。
