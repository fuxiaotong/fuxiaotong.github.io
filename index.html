vim 是一个非常强大的工具，因为可以安装各种各样好用的插件，vim 和 emacs 被成为程序员们的最爱，这里我来简单介绍一个如何优雅地使用vim

snippet

mru

在vim中输入 :MRU 我们可以浏览到之前打开的文件，快速切换到之前的文件

NERDTree

在vim中输入 NERDTree 我们可以在右边浏览到目录树，是一个非常方便的功能

cscope

下面介绍一个非常实用的工具 cscope
这个工具的作用是实现函数的跳转，如果你是ubuntu系统，请执行
apt-get install cscope
首先在目录下使用“cscope -Rkqb”生成cscope.out文件
-R ：表示包含此目录的子目录，而非仅仅是当前目录；
-b ：此参数告诉cscope生成数据库后就自动退出；
-q ：生成cscope.in.out和cscope.po.out文件，加快cscope的索引速度可能会用到的其他选项：
-k ：在生成索引时，不搜索/usr/include目录；
上面一步就是为你的源文件建立一个cscope数据库
在Vim启动之后手动地添加这个数据库，输入: cs add cscope.out 即可，然后你可以通过:cs show来检查一下是否被添加进去
接下来输入:set cscopetag 你可以把光标移动到某个位置通过Ctrl+]可以直接跳转到定义处，通过Ctrl+t可以返回跳转前的地方
此外还可以调“cscope find”（或 cs f ）命令进行查找

还有一点：默认情况下cscope值会在当前目录下针对c、iex和yacc（扩展名分别为.c、.h、.I、.y）程序文件进行解析（如果指定了-R参数则包含其自身的子目录）。这样出现的问题就是，我们对于C++或Java文件怎么办，解决方案是：我们可以生成一个名为cscope.finds的文件列表，并交由cscope去解析。在Linux系统中，生成这个文件列表的方法是：
find . –name "*.java" > cscope.files
然后运行cscope –b 命令重新生成数据库就OK了.

python之旅

python 印象

python给我的第一印象是一个非常方便的语言，它似乎能尽可能缩短你的代码量，让你更加专注于你的逻辑，让你能更加轻松更加容易地实现你的idea。有了这个神器，你能快速完成一个用C++、java这些语言很难完成的任务，话不多说，让我们进入这个神圣的殿堂，翻开python的第一页书吧！

入门

我们来看看 python 的操作符，print 3**2 结果是9,也就是说两个星号表示的是乘方
print 2 > 4 返回的是False
在表示不等的时候最好用!=
python 只能是以字母或者下划线结尾的，也是大小写敏感的。python也是一门动态语言，也就是说，它不需要事先声明变量的类型，而是在赋值的那一刻进行初始化的。

操作系统学习

操作系统

对于非计算机专业或者计算机软件专业但是操作系统学得不好的人那怎么办呢，其实现在学习计算机知识的门槛并不高，可以去 学堂在线 或者 coursera 搜索操作系统的课程学习，个人比较推荐的是 学堂在线 的操作系统课程
其中列出一些个人认为比较重要章节，必须要掌握：
物理内存管理：连续内存分配（连续内存分配、伙伴系统）
物理内存管理：非连续内存分配（页表、快表、多级页表、反置页表、段页式存储管理）
虚拟存储（交换、虚拟页式存储、缺页异常）
页面置换算法（最优、先进先出、最近最久未使用）
进程和线程（进程控制块、进程状态切换、线程概念、用户线程、内核线程）
进程控制（进程切换、创建、加载）
处理机调度（先来先服务、短进程优先和最高响应比优先调度算法、时间片轮转、多级反馈队列）
同步互斥
信号量与管程
死锁与进程间通信（死锁的四个必要条件、银行家算法、信号和管道、消息队列和共享内存）
文件系统（文件描述符、虚拟文件系统、文件缓存、空间空间管理和冗余磁盘阵列RAID）
